PnPMusic
========

This is a program for gamemasters to put their music into different categories and let them be played randomly by category.

## Upcoming Features

- Better categorising
- Soundeffects
- A nicer GUI

## Example Music
- Cowboy Sting 
- Run Amok
- Undaunted
- Winner Winner
- Guts and Bourbon
- Past the Edge
- Reign Supreme
- Drums of the Deep
